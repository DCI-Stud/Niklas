Grundlagen in HTML & CSS

3 Wege zum benutzen von CSS

Inline-Styling: Hierbei wird das CSS direkt in das HTML-Element eingefügt, indem das style-Attribut verwendet wird. Dies ist jedoch nicht die beste Methode, da sie die Wartung erschwert und die Trennung von Inhalt und Design aufhebt

Interne Stylesheets: Hierbei wird das CSS innerhalb des style-Tags im head-Bereich der HTML-Datei geschrieben. Dies ist nützlich, wenn Sie spezifische Stile für eine einzelne Seite haben

Externe Stylesheets: Hierbei wird das CSS in einer separaten Datei (mit der Endung .css) geschrieben und dann in die HTML-Datei eingebunden. Dies ist die gebräuchlichste Methode, da sie eine klare Trennung von Inhalt und Design ermöglicht und es einfacher macht, den Stil auf mehrere Seiten anzuwenden
