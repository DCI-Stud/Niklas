Grundlagen in HTML & CSS

Was ist CSS und wofür braucht man es?

  • CSS ist die Abkürzung für Cascading Style Sheets. (Cascade - Wasserfall)
  • CSS stellt den style eines HTML-Dokuments dar.
  • CSS beschreibt, wie HTML-Elemente auf dem Desktop, auf Smartphones, Tablets, Papier oder in anderen Medien dargestellt werden.
  • Mit CSS kann das Layout mehrerer Webseiten gleichzeitig gesteuert werden.
  • “Cascading” bedeutet: Auf ein HTML-Element kann mehr als eine Regel angewendet werden. Die letztlich verwendete Regel wird von allgemeineren Regeln herunter zur spezifischen Regel ausgewählt. (Cascade - Wasserfall
