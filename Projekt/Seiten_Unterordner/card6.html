Was ist die Flexbox?

Die Flexbox, oder genauer gesagt das CSS Flexible Box Layout, ist ein wichtiges Werkzeug in CSS, das es ermöglicht, flexible und responsive Layouts zu erstellen

Es funktioniert intelligenter und dynamischer als traditionelle Layoutmethoden: Das Layout passt sich flexibel an das verwendete Display an1. Platz wird aufgefüllt oder Elemente enger zusammengeschoben, damit alles sichtbar bleibt

Flexbox arbeitet mit zwei Achsen: Die Hauptachse verläuft typischerweise horizontal, die Kreuzachse vertikal1. Anhand dieser Achsen können die Elemente innerhalb der Box angeordnet werden1. So lassen sich Elemente in Bezug zueinander verteilen

Flexbox basiert auf einem Container (flex container), in dem wiederum mehrere Elemente (flex items) untergebracht sind1. Der Container gibt seine Eigenschaften an die Elemente ab

Es ist eine moderne und einfache Möglichkeit, responsive und flexible Layouts zu erstellen, ohne feste Größenangaben und weitere CSS-Einstellungen wie position, float oder clear nutzen zu müssen2. Dabei kann nicht nur die Größendarstellung, sondern auch die Reihenfolge der Elemente unabhängig vom HTML-Code durch CSS festgelegt werden
