Grundlagen in HTML & CSS

Was ist die Syntax?

Die Syntax für HTML und CSS ist recht einfach zu verstehen. HTML (HyperText Markup Language): HTML wird verwendet, um die Struktur einer Webseite zu erstellen. Es besteht aus verschiedenen “Tags”, die den Inhalt auf der Seite kategorisieren.

CSS (Cascading Style Sheets): CSS wird verwendet, um das Aussehen einer Webseite zu gestalten. Es ermöglicht Ihnen, Stile wie Farben, Schriftarten, Layouts und mehr auf HTML-Elemente anzuwenden.

Externe Stylesheets: Hierbei wird das CSS in einer separaten Datei (mit der Endung .css) geschrieben und dann in die HTML-Datei eingebunden. Dies ist die gebräuchlichste Methode, da sie eine klare Trennung von Inhalt und Design ermöglicht und es einfacher macht, den Stil auf mehrere Seiten anzuwenden
