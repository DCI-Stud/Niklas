Was ist der Nachbarselektor?

Nachbarselektor: Ein Nachbarselektor, auch bekannt als angrenzender Geschwisterselektor, in CSS wird verwendet, um ein Element auszuwählen, das unmittelbar nach einem bestimmten Element steht.

Die Syntax für einen Nachbarselektor ist Element1 + Element2. Hierbei steht das +-Zeichen für die Nachbarschaftsbeziehung zwischen den Elementen.
