Grundlagen in HTML & CSS

Was ist die letzte Version von HTML?

HTML Version: Die letzte Version von HTML ist HTML 5.2, die seit dem 14. Dezember 2017 existiert1. Sie wird bereits von vielen aktuellen Webbrowsern und anderen Layout-Engines unterstützt

Es ist jedoch wichtig zu beachten, dass die Web Hypertext Application Technology Working Group (WHATWG) vorgeschlagen hat, dass HTML5 die letzte Version von HTML sein soll. Zukünftige Änderungen werden in einem “Living Standard” ohne weitere Versionsnummern weiterentwickelt
