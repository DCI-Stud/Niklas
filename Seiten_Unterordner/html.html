Grundlagen in HTML & CSS

Was bedeutet HTML und wofür wird es gebraucht?

  • HTML (HyperText Markup Language) ist eine Auszeichnungssprache zur Strukturierung von Webseiten.
  • Mit HTML werden Inhalte wie Bilder, Texte und Tabellen strukturiert.
  • Es besteht aus Tags, die den Inhalten bestimmte Aufgaben zuweisen, z.B. für Absätze.
  • HTML ist die Grundlage des Internets und wird von Webbrowsern wie Chrome und Firefox gelesen und grafisch dargestellt.
